How to get an agent reference number for PAYE

Find out how to get an agent reference number for PAYE, Construction Industry Scheme (CIS) or Employment Related Securities (ERS).

From:
HM Revenue & Customs
Published
7 September 2022

Contents

  1. What you’ll need
  2. Get an agent reference number

The PAYE (Pay As You Earn), Construction Industry Scheme (CIS) and Employment Related Securities (ERS) services are all linked in the HMRC online services account.

If you get an agent reference number and enrol one of these services, the others will automatically be added to your account. If you remove any one of these linked services, they will all be removed from your account and you will have to add them again.

What you’ll need

To get an agent reference number for PAYE, CIS or ERS you need to apply for one online.

You’ll need your:

  • full name
  • trading name (if different)
  • telephone number
  • email address

Get an agent reference number

Use the online service to get an agent reference number.

When you’ve got your agent reference number

Once you’ve got your agent reference number, sign into HMRC online services to enrol the tax services you want onto your account.

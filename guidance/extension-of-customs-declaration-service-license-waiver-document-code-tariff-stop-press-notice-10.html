Extension of Customs Declaration Service license waiver document code (Tariff stop press notice 10)

Find out about the extension Import Tariff measures and commodity codes for cat and dog fur, organics and mercury.

From:
HM Revenue & Customs
Published
1 September 2022

The Customs Declaration Service license waiver document code 999L has been extended to import tariff measures and commodity codes for cat and dog fur, organics and mercury.

We will add document waiver code 999L to further import and export measure types throughout August 2022.

These further updates will include:

  • import controls — on fluorinated greenhouse gases, seal products, waste
  • export controls — on cat and dog fur and dual use goods

More information

To stop getting the Tariff stop press notices, or to add recipients to the distribution list, email: tariff.management@hmrc.gov.uk.

See the UK Trade Tariff for the full duty rate, unit of quantity, preferences, quotas, footnotes and VAT rate.

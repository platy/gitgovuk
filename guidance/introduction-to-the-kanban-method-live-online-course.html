Course summary

Kanban is an agile method used widely across both the public and private sectors.

This course is an opportunity to learn more about how teams can use Kanban when delivering services across government. You will learn about the way in which Kanban should be used, how it supports teams meeting the Service Standard and why it is effective in promoting flow and removing blodkers from teams’ workstreams.

This course is run online. Attendees will a computer with broadband that can access Zoom video conferencing, with microphone enabled. If you want to, you can join using a personal device to join. All content is unclassified, and you won’t need to download any software.

Course aims and objectives

This course will enable you to:

  • identify the key blockers to maintaining a sustainable development pace
  • list the Kanban principles and practices
  • define the key signs of inefficiency in Kanban
  • list the key points for successfully implementing Kanban
  • explain the key cultural needs to enable Kanban to work

Eligibility

Our courses are available to civil servants, local government employees, devolved administrations and other public servants, at any grade.

Duration

The course takes 2 hours, including an introduction to the online tools.

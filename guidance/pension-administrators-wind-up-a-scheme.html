Guidance

Wind up a pension scheme

Submit an Event Report to HMRC to declare that a scheme has ceased to exist.

Winding up a pension scheme means the scheme ceases to exist.

The scheme assets will either be:

  • transferred to other pension schemes
  • used to buy annuities to provide the members with their benefits

How to declare that a scheme has ceased to exist

If you’re a scheme administrator you must submit an Event Report to HMRC and provide the date the scheme completed winding-up.

You must submit the Event Report within 3 months of the scheme wind up completing.

You must continue to meet all information reporting requirements until you tell HMRC of the wind up.

If your Pension Scheme Tax Reference (PSTR) begins with ‘0’

If the pension scheme has a PSTR beginning with ‘0’ you’ll need to submit an Event Report on the Pension Schemes Online service.

If the scheme has been migrated to the Managing Pension Schemes service, you must also email pensions.administration@hmrc.gov.uk with ‘MPS scheme wind up’ in the subject line and include the:

  • PSTR
  • name of the pension scheme
  • scheme administrator ID
  • date of the wind up

If your PSTR begins with ‘2’

If the pension scheme has a PSTR beginning with ‘2’ you must email pensions.administration@hmrc.gov.uk with ‘MPS scheme wind up’ in the subject line and include the:

  • PSTR
  • name of the pension scheme
  • administrator ID
  • date of wind up

Information you’ll need

To complete the Event Report you’ll need the:

  • Government Gateway user ID and password you used when you registered for the Pension Scheme Online service
  • PSTR of the scheme
  • date the scheme completed winding-up

If you do not have the PSTR you’ll need the pension scheme name and either the SF reference number (for schemes approved before 6 April 2006) or the policy number.

If you are a practitioner filing the Event Report for a scheme administrator you will also need the name, address and scheme administrator ID of the scheme administrator who has approved the content of the Event Report.

If the scheme was approved before 6 April 2006

Most schemes approved before 6 April 2006 have been given a PSTR. If you have not needed to contact HMRC you probably will not have signed up to use Pension Schemes Online or have the scheme’s PSTR.

You can use the scheme’s tax approval reference number to find the scheme record on Pension Schemes Online. The tax approval reference number will start with SF, for example SF 000/000000/000000/A.

If you’ve got the tax approval reference number, contact the Pension Schemes Services Helpline to check it’s the right number and that it’s complete.

If you have not got the reference number you’ll need to get it from the pension provider or insurance company.

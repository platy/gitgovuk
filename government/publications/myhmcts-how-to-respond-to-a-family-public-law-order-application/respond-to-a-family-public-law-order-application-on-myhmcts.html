1. Before you start

Before you can use public family law services on MyHMCTS, you and/or your organisation must have completed other steps:

  1. Set up HMCTS Payment by Account (so that you can pay any relevant fees)
  2. Registered your organisation with MyHMCTS
  3. Set up your MyHMCTS user account

Please read the MyHMCTS registration guidance for further information and instructions. If your organisation has an account already, contact the administrator to arrange for your account creation.

Once you have a MyHMCTS account, you can sign in using your email address and password.

If you need additional help with MyHMCTS email MyHMCTSsupport@justice.gov.uk. We aim to respond within 5 working days.

If you are nominated as the respondent solicitor but your case administrator cannot find the case, it means either:

  • the case has not yet been served on you, or
  • the applicant was not able to add your details to the case to give you digital access to the file

In these circumstances, contact the applicant local authority or representative and provide them with the information they need to add you to the case.

2. Sign in and find a case

1. Sign in to MyHMCTS with your account and password details.

2. You will see the home screen, known as the ‘Case List’. Use the filters to find and view applications that have been issued. You may need to reset the filters to find the case. If you still cannot see the case, contact your case administrator. If after that you are still having issues, contact us for help.

3. Select the appropriate issued application to see the case management screen. The ‘Documents’ tab will be selected by default.

The tabs you see will include:

  • Documents – including statements and the C110a
  • Hearings – all hearing details
  • Draft orders – including pre-hearing and post hearing when waiting for the judge to approve the order (will only show if there is a draft order to approve)
  • Orders – approved orders (a notification will be sent when an order is approved)
  • People in the case – details of each party in the case
  • Legal basis – threshold criteria, allocation decisions and application summary
  • Court bundle – this tab will be hidden until a bundle is uploaded
  • Other applications – additional applications made within proceedings

3. Manage documents and orders

Upload a document

1. Select ‘Manage documents’ from the next step dropdown and then select ‘Go’.

2. Select a document category and then ‘Continue’.

3. Select a document type. If it relates to a hearing, select the appropriate hearing. Then select ‘Continue’.

4. Name the document and browse to upload it. Select the appropriate type of document and then ‘Continue’.

5. Check your answers. If they are correct, select ‘Save and continue’. The document will appear in the documents tab.

Replace a document

You can replace a document if you have uploaded it in error or need to amend it.

1. Select ‘Manage documents’ from the next step dropdown menu.

2. Repeat steps 2 and 3 from Upload a document – make sure to choose the same category, type and hearing as required.

3. In the upload document screen, check that the document name is the same as the file you want to replace. Browse and select the new file and then ‘Continue’.

4. Check your answers are correct, with the new version of the document added. Then select ‘Save and continue’.

Remove a document

You can remove a document if you have uploaded it in error or no longer wish to add it to the case.

1. Follow the same steps to replace a document then select ‘Remove’ within the ‘Supporting documents’ section.

2. You will be asked if you’re sure you want to remove the document. Select ‘Remove’ then ‘Continue’.

3. Check your answers to make sure the document is no longer listed. Then select ‘Save and continue’.

The document will no longer appear in the ‘Documents’ tab.

Upload a draft order before a hearing

1. Select ‘Upload draft orders’ from the next step dropdown, then select ‘Go’.

2. Select ‘Case Management (CMO)’ from the order options, then ‘Continue’.

3. Select ‘Draft CMO’ then choose which hearing the order relates to from the dropdown menu. Then select ‘Continue’.

4. Browse for the file that you want to upload as the draft order. Then select ‘Continue’.

Do not add additional orders in the ‘Case summary and supporting documents’ section – they cannot be approved if uploaded here.

5. The next screen will show the name of the judge that the order will be sent to for approval. Select ‘Continue’.

6. Check your answers are correct. If you need to amend any of the details, select ‘Change’ next to the appropriate detail. When all the answers are correct, select ‘Submit’.

The hearing judge will receive a notification that the draft order is available to view. This is not the order they will approve.

Upload an agreed order for a judge to approve

1. Select ‘Upload draft orders’ from the next step dropdown, then select ‘Go’.

2. Select the relevant order option. If you have more than one order relating to the same hearing, select both options – for example, if you want to upload a police disclosure order.

3. Select ‘Agreed CMO’ then choose which hearing the order relates to from the dropdown menu. Then select ‘Continue’.

4. Browse for the file that you want to upload as the agreed order. Make sure the file is named ‘Final order’ not ‘Draft order’. Then select ‘Continue’.

Do not add additional orders to the ‘Case summary or supporting documents’ section. You can add additional orders on the next screen.

5. Name and browse for an additional order, such as a police disclosure order. When you have added all the additional orders, select ‘Continue’.

6. The next screen will show the name of the judge that the order(s) will be sent to for approval. Select ‘Continue’.

7. Check your answers are correct. If you need to amend any of the details, select ‘Change’ next to the appropriate detail. When all the answers are correct, select ‘Submit’.

The hearing judge will receive a notification that the agreed order(s) is available to view.

4. Make an additional application

You can make an application for additional orders or provisions to be made in the case.

If the application is confidential and not to be made available to the other parties, you must send it to the court on paper rather than on MyHMCTS.

1. Select ‘Upload additional application’ from the next step dropdown. Then select ‘Go’.

2. Select the type of application. If you are making a C2 application to add or remove someone on a case, state whether it is by consent or without notice. Then select the relevant applicant from the dropdown menu and ‘Continue’.

If you have selected ‘Other specific order’, go to step 4. If you have selected a C2 application or both options, go to the next step.

3. If making a C2 application, browse for and upload the application. Select any of the relevant check boxes below the question ‘Are you using the C2 to apply for any of the below?’ If you check ‘Parental responsibility’, select who is seeking responsibility. Then select ‘Continue’.

4. Select the type of other application you are making and upload the document. If you are making a C1 with supplement application, select the supplement name and upload the document. Select ‘Continue’.

5. Select whether to notify all people on the case about the application. If you select ‘No’, choose who should be notified. Then select ‘Continue’.

6. On the payment screen, select ‘Yes’ if you have a PBA account and complete the account details. Then select ‘Continue’. Only select ‘No’, if you do not have a PBA account – the court will contact you for payment

Make sure to complete the customer reference – this will help your finance team reconcile the account when they receive the invoice.

7. Check your answers are correct. If you need to amend any of the details, select ‘Change’ next to the appropriate detail. When all the answers are correct, select ‘Save and continue’.

You will find the application in the ‘Other applications’ tab.

The service will notify the CTSC, who will refer the application to the allocated judge.

5. Get help and support

For help with registration and managing your organisation, email MyHMCTSsupport@justice.gov.uk. We aim to respond within 5 working days.

For help with technical issues using MyHMCTS to manage a family public law case, email PublicLawAndAdoption@justice.gov.uk.

For support or guidance on a case, email contactfpl@justice.gov.uk.

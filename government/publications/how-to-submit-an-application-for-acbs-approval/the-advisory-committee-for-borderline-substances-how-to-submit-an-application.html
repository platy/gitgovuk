Send an application

Print and post your application to:

ACBS Secretariat
Desk 2NE007
Department of Health and Social Care
Quarry House
Leeds
LS2 7UE

Hardcopy applications must be bound using wire-o binding (wiro binding), otherwise known as twin loop wire or double loop wire binding. The application will otherwise not be accepted.

You must also send a PDF copy of the application to acbs@dhsc.gov.uk. The PDF copy must be a single file PDF document and be identical, page-for-page, to the hardcopy. All information must be legible and any product labels or images must be of a high resolution.

Please ensure you read the following additional rules and caveats in respect of type 1, type 2 and type 3 applications.

Type 1 and type 2 applications

Type 1 applications are for new formulations that have substantiated advantages and where there is no comparable product on the market (Part XV of the Drug Tariff).

Type 2 applications are for formulations that are broadly similar in composition to existing products already on the market.

Type 1 and type 2 applications are reviewed at ACBS meetings. Please send 4 hardcopies of your application and a PDF copy. Both the hardcopies and the PDF copy must be received by the ACBS Secretariat on or before the closing date for the relevant meeting.

Type 3 applications

Type 3 applications are for changes to existing products that have already been approved by the ACBS.

These are reviewed throughout the year and may be submitted at any time. Please only send a PDF copy of the application - hardcopies are not required unless requested by the Secretariat. In exceptional circumstances, we may review a type 3 application and decide that it needs to be reviewed at a committee meeting. You will be notified if we decide this.

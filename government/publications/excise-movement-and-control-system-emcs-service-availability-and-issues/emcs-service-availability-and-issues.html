End of EU Exit transition on 31 December 2020

As a result of EU Exit transition ending on 31 December 2020, there are some changes to how goods can move under Excise Duty Suspension and changes have been made to the EMCS system to reflect the changed situation.

You can find more information on:

  • importing excise goods to the UK from the EU
  • exporting excise goods to the EU
  • moving excise goods as freight under the Northern Ireland Protocol from 1 January 2021

The excise notices and other related guidance on GOV.UK have also been updated.

Service issues

XIWK-warehousekeeper and HMRC portal issue when submitting a draft eAD for a dispatch from a warehouse in Great Britain

We have released a fix for the HMRC portal issue which began on 1 March 2022.

Excise warehousekeepers with an ID starting ‘XIWK’, who have warehouse premises in Great Britain (England, Scotland and Wales), can now submit draft electronic administrative document (eAD) movements to EMCS for their premises through the HMRC portal.

You should now use the Competent Authority Dispatch Office ‘GB004098’ for a movement being dispatched from a location in Great Britain.

Any movements which were dispatched under the EMCS fallback procedures should be entered into EMCS as soon as possible. You should tick the ‘deferred movement’ box so you can enter the correct date of dispatch, which will be in the past.

We apologise for any inconvenience this issue may have caused.

Update — EMCS rejected exports matching issue of 9 February 2022

We have released a fix for the EMCS export matching issue which began on 9 February 2022.

The backlog of CHIEF matching messages has been downloaded to EMCS and has finalised most of the affected movements.

If any of the export eAD movements are still showing as undischarged, you should email emcs.helpdesk@hmrc.gov.uk with the Administrative Reference Code (ARC) number and the corresponding export entry details.

If EMCS has issued a report of receipt to finalise an export movement, any prior or subsequent export rejection messages from EMCS can be ignored.

In some instances, EMCS is incorrectly issuing export rejection messages after an export ARC number has already been finalised. This will continue to happen for a number of weeks until a backlog in a secondary EMCS processing queue is cleared.

Northern Ireland traders exporting goods from Northern Ireland warehouses

When an XI-ARC is declared in Box 40 of the CHIEF export entry, the following error code message will be displayed: E3392 — ‘ARC Ref 3rd — 4th chars do not identify an EU Member State’

If you use an export agent, make them aware of the following workaround:

  1. Enter the local reference number (LRN) in Box 40 of the customs declaration as if the EMCS movement had been dispatched under the fallback procedure. For example, Z (first field)/FAD (second field)/LRN (third field). The LRN should be the same as the one submitted on the EMCS movement.

  2. Declare the EMCS XI-ARC number in Box 44 (the additional information field). For example, ‘EMCS ARC number; 21XI00000000012345678’.

  3. Declare the premises excise reference number (ERN) in Box 49, in the GB-format, not XI. For example, XI00001234567 should be entered GB00001234567. If you use the XI ERN, the following error code message will be displayed: E1799 — the premises of the authorising country must be in the EU.

As a result of this issue, CHIEF will not discharge the XI-ARC movement on EMCS.

Once the consignment has been exported and the CHIEF export entry has reached departed status, you should submit evidence of export to emcs.helpdesk@hmrc.gov.uk to request the closure of the undischarged ARC, as per the existing procedures in Notice 197, Section 13.10.

Details

This note sets out the methodology used to allocate the £2.6 billion UK Shared Prosperity Fund (UKSPF) across the UK up to the end of the 2024-25 financial year.

UKSPF allocation model

The UKSPF model shows the steps taken in using data to allocate the UK Shared Prosperity Fund.

UKCRF index model

The UK Community Renewal Fund (UKCRF) index model shows all the steps taken in using data to update the UK Community Renewal Fund Index. The UKCRF index is used as part of a formula to allocate funding to places as part of the UK Shared Prosperity Fund.

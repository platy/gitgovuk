College financial benchmarking tool: guidance

HTML

Details

Users can filter the benchmark population based on certain criteria. The tool provides a number of benchmarking reports, trend analysis and access to the underlying datasets.

The latest version of the tool includes data submitted in colleges’ finance records for the year:

  • 2020 to 2021

The tool for the prior year, which remains on this webpage, includes data submitted in colleges’ finance records for the years:

  • 2018 to 2019
  • 2019 to 2020

Additional information is set out in our college accounts direction.

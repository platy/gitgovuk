Details

LRS registration form

Use this form to gain access to the LRS portal and services

Learner registration bodies (schools, academies, FE providers, Local Authorities, HE providers) users need to submit this along with the learning provider agreement (LPA).

Awarding organisations need to submit this along with the awarding organisation agreement form.

Change of super user form

Used if an organisation wishes to amend their super user.

Find the guides on how to use the LRS on the User guides and lists for LRB users and for AO users page

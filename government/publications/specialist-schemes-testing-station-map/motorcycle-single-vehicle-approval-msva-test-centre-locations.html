This is a list of test centres that carry out Motorcycle Single Vehicle Approval (MSVA).

You need to say which of these centres you want to use when you apply for an MSVA test.

London and the South East

There are 3 test centres in London and the South East:

  • Gillingham
  • Southampton
  • Yeading

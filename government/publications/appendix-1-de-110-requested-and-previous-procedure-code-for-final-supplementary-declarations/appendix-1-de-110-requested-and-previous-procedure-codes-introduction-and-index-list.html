This Appendix only contains the additional notes on specific declaration data elements in relation to the 4-digit Procedure Code and does not constitute full guidance on how the FSD should be completed.

The completion notes in this Appendix are a supplement to the main CDS Declaration Completion Instructions for Final Supplementary Declarations.

Where the DE 1/10 completion notes contain different instruction from the main Declaration Completion Instructions or other Appendices, the completion instructions in Appendix 1 take precedence.

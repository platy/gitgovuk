Who should read

All ESF beneficiary organisations, European Social Fund Division and Greater London Authority

Purpose

This action note is to notify CFOs and Grant Recipients that the ESF Dual Forecast and Submission form has been updated to make it suitable for use with 2020 claims.

Action

ESF Projects should use this new version of the form for all claims from Q1 2020 onwards.

As always, ESF Projects should ensure that their ‘Actual’ and ‘Contracted’ figures in each tab within the form are as accurate as possible as this information will contribute to the Managing Authority’s consideration of performance for each project.

If a project is submitting a claim relating to 2019 or earlier, the previous version of the form should be used. This version is only applicable to claims for Q1 2020 onwards.

Finally, please note that the ESF Managing Authority is currently undertaking a review of this product, this may result in a re-design and/or replacement product at a future point. The outcome of this review and any resulting changes will be communicated separately to Grant Recipients.

If you have any questions about this Action Note please contact esf.2014-2020@dwp.gov.uk

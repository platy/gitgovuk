This Appendix contains the specific completion rules instructions for the 7-series Additional Procedure Codes for Data Element (DE) 1/11 for Inventories Imports.

Use the index list and the Additional Procedure Codes completion notes to help you identify the correct code to use and for guidance on how to complete the declaration.

Notes:

The completion notes in this Appendix are a supplement to the main UK Trade Tariff, CDS Volume 3 Inventory Import Declaration Completion Guide.

This Appendix only contains any additional completion notes for each 3-digit Additional Procedure Code and does not constitute full guidance on how the declaration should be completed.

Only one 4-digit Procedure Code can be declared against each goods item in DE 1/10, however, this may be tailored to the specific circumstances for the goods treatment by the use of multiple Additional Procedure Codes in DE 1/11.

These completion notes are a supplement to the DE 1/10 Procedure Code Notes and therefore any instructions relating to the requested or previous procedure included against the relevant DE 1/10 Procedure Code (see Appendix 1: DE 1/10: Requested and Previous Procedure Codes) must also be followed.

The DE 1/10 completion notes, including conditions and restrictions, are not repeated within or superseded by the DE 1/11 completion notes.

71M: Movements: in Temporary Storage (MiTS)

Description of Procedure:

Customs Clearance Request (CCR) for non-Union goods being moved between approved Temporary Store (TS) facilities using Movements in Temporary Storage (MiTS) procedures.

Goods covered

Inter shed removals by parties authorised under the TS arrangements to move goods between approved TS facilities, where this cannot be completed by direct inventory messaging.

Conditions for Use:

This Additional Procedure Code may only be used:

  • by authorised Temporary Storage Operators
  • for goods already presented to customs and entered to TS
  • to move goods between approved TS facilities

A guarantee must be provided to secure the goods during the movement.

Unless otherwise stipulated, the dispatching facility remains liable for the goods and any incurred customs debt until the movement is discharged appropriately on arrival at the receiving facility (Article 193 (4), Commission Implementing Regulation (EU) No. 2015/2447 (IA)).

Please refer to Notice 199A for additional conditions relating to MiTS.

Restrictions on usage:

Where MiTS have been pre-authorised to be undertaken using messages within inventory systems, this Additional Procedure Code should not be used.

This Customs Clearance Request does not extend or re-start the 90-day limit.

Notes:

Goods may remain under TS for a maximum period of 90-days. Goods must be entered to a customs procedure or re-exported within 90-days.

Failure to enter the goods to a customs procedure or re-export the goods within the 90-day period will result in a demand for any customs duties and other charges. The TS Operator may also be subject to a customs civil penalty.

Where goods are being moved to an External Temporary Storage Approved Depository, Additional Procedure Code 74A must also be declared in DE 1/11.

Additional Procedure Code 71M can only be used with Requested and Previous Procedure code (DE 1/10): 0007.

72M: Movements: in Records for Special Procedures

Description of Procedure:

Customs Clearance Request (CCR) for the movement of non-Union goods from another Member State using Special Procedures: Movements in Records.

Goods covered

This Additional Procedure Code may be used to release goods from an inventory system where they are being moved under a Special Procedure using Movement in Records procedures from an authorised party in another Member State.

Conditions for Use:

This Additional Procedure Code may only be used to release already entered to the Special Procedure in another Member State.

A full authorisation for a Special Procedure must be held to release goods using this Additional Procedure Code.

Special Procedure movements can only be made:

  • Between different storage facilities designated in the same authorisation;
  • From the customs office of placement to the storage facilities, or;
  • From the storage facilities to any customs office indicated in the authorisation for a special procedure, empowered to release goods to a subsequent customs procedure (as referred to in Article 211(1) of the Code).

Restrictions on usage:

This Additional Procedure Code may not be used to place non-Union goods into a Special Procedure. They must have been prior entered to the Special Procedure in the other Member State.

Where any of the Special Procedure movement conditions (see above) cannot be met, a written customs declaration will be required and this Procedure Code may not be used.

This Additional Procedure Code may not be used to move goods between Customs Warehouses.

A written Customs Declaration is required for goods moving between customs warehousing premises in different Member States.

Notices:

Where Movements in Records: Special Procedures is being made using EIDR, Additional Procedure Code 73N must also be declared in DE 1/11.

Additional Procedure Code 72M can only be used with Requested and Previous Procedure code (DE 1/10): 0007.

73N: Notification of Presentation (NoP) under EIDR

Description of Procedure:

Customs Clearance Request (CCR) for goods declared to a customs procedure using Entry in Declarantâs Records (EIDR) where a Notification of Presentation (NoP) is required.

Goods covered

This Additional Procedure Code may be used to present goods to Customs to request release where EIDR is used to declare the goods to a customs procedure.

Conditions for Use:

This Additional Procedure Code may be used where a NOP is being made under the EIDR procedure for:

  • goods declared to a customs procedure using EIDR in another Member State
  • goods being moved under Special Procedures using EIDR and Movement in Records

Please see Notice 760 for additional conditions relating to EIDR and Notifications of Presentation (NoP).

DE 3/41 must be completed with the EORI of the person presenting the goods to customs at the border. This does not have to be the EIDR authorisation holder.

Restrictions on usage:

Where the goods are being released direct to a customs procedure or use at the border:

The Procedure Code and Additional Procedure Codes used in DE 1/10 and DE 1/11 must be selected from Appendices 1 and 2 from the UK Trade Tariff, CDS Volume 3, Imports Tariff and this Additional Procedure Code must not be used.

74A: Approved Depository

Description of Procedure:

Customs Clearance Request (CCR) for the movement of goods under TS to an Approved Depository.

Goods covered

This Additional Procedure Code is to release applicable non-Union goods under TS for movement to an External Temporary Storage Facility Approved Depository (ETSFAD) premises.

Conditions for Use:

This Additional Procedure Code may only be used to request release of personal effects and household goods.

Please refer to Notice 199A and Notice 199B for additional conditions relating to Approved Depositories.

A guarantee must be provided to secure the goods during the movement.

Unless otherwise stipulated, the dispatching TS facility remains liable for the goods and any incurred customs debt until the movement is discharged appropriately on arrival at the receiving ETSFAD (Article 193 (4), Implementing Regulation (EU) No. 2015/2447 (IA)).

Additional Procedure Code 71M must also be declared in DE 1/11.

Restrictions on usage:

This Additional Procedure Code may not be used to release goods to an External TS facility other than an ETSFAD.

This Additional Procedure Code may not be used to request release of commercial goods.

Notices:

Notice 199A: temporary storage and approved depositories

Notice 199B: approved depositories under external temporary storage facility

Specific fields in the declaration/notes on completion

Notes:

An External Temporary Storage Facility Approved Depository (ETSFAD) serves the same function as a temporary storage facility and is subject to the same rules and regulations, however, an approved depository can only be used for the storage of personal and household effects.

Additional Procedure Code 74A can only be used with Requested and Previous Procedure code (DE 1/10): 0007.

There are now a sizeable number of evaluation courses. We have listed some in this document, as a service to the evaluation community. Note that we have not necessarily tested all these courses.

If you run or have been on a good evaluation training programme that you would like to list here, please contact etf@cabinetoffice.gov.uk.

Information and resources on this page are for FMS software providers only.

Academy trust finance automation: getting started

This guide:

  • provides FMS suppliers with quick links to register their software, access the developer hub and test environments
  • gives detail on the process for updating suppliers with the most up to date academies chart of accounts (CoA)
  • provides links to CoA versions specific for supplier use which will be uploaded onto this page in March and August
  • for the most recent chart of accounts available to trusts, users should visit Academies chart of accounts and automating the accounts return webpage

Guide to software integration with automation API

Provides detailed technical requirements that FMS suppliers need to know to successfully automate their trusts chart of accounts data into the DfE’s financial returns.

Academies chart of accounts 2021-22 - for 2021-22 AR Automation - March 22

This spreadsheet includes all known updates from the CoA 2020-21, and is available for the initial testing by FMS suppliers of the 2021-22 automated accounts return. A final version will be released in August 2022.

Trusts who wish to find out about the CoA should visit the Academies chart of accounts and automating the accounts return webpage.

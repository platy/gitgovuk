Purpose of this guidance

This is a step-by-step guide on how to register multiple PCR and rapid lateral flow test kits at once on the organisation registration portal.

It covers:

  • the benefits of using multiple upload to register multiple test kits at once
  • the information you’ll need to have to hand before you begin
  • how to download the multiple upload spreadsheet, formerly known as the record-keeping spreadsheet
  • how to fill in the multiple upload spreadsheet before testing
  • how to upload the multiple upload spreadsheet to register or report test kits

A new version of the spreadsheet was introduced on 19 May. You can now:

  • input the test details at the beginning and directly on to the spreadsheet, saving time
  • input both PCR and rapid lateral flow tests into the spreadsheet
  • if registering a rapid lateral flow test, you’ll be able to add the result in the spreadsheet too
  • check all the details before submitting, using the updated review page

You must always use the most up-to-date version of the multiple upload spreadsheet

The benefits of using the multiple upload feature to register test kits

User research has shown that registering each PCR test kit one at a time is a time-consuming and resource-intensive process.

Multiple registration upload (bulk registration) has been developed to address this issue by allowing you to register up to 100 test kits at once.

Organisations are now able to register PCR and rapid lateral flow tests, and report rapid lateral flow test results, using a single spreadsheet. To do this, you will need to:

  • download the updated spreadsheet – do not add any sheets, rows or columns
  • enter details for up to 100 tests at a time
  • use separate spreadsheets for staff and residents
  • save the completed spreadsheet then upload it

When performed correctly, multiple registration:

  • significantly speeds up the registration process
  • significantly reduces the administration effort required to register test kits
  • gives those registering test kits significant time back to complete other tasks

What you’ll need to register test kits online

1. You’ll need your unique organisation number (UON).

This is 8 digits long, unique to your organisation and is used to access all the National Testing Programme’s online services. All users of the organisation registration portal will have been sent their UON from organisation.coronavirus.testing@notifications.service.gov.uk.

If you need to be reminded of your UON, you can visit the UON lookup or contact the National Testing Programme Contact Centre on 119.

2. You’ll need the personal information of all those who you are registering test kits for, including:

  • first and last name of staff member/non-staff member
  • date of birth
  • gender
  • ethnic group
  • first line of address, country and postcode – for staff this should be their home address, for non-staff resident at your organisation this should be the organisation address
  • NHS number
  • working circumstances, area of work, occupation and employer
  • email of the person who should receive the test result
  • phone number of the person who should receive the test result
  • symptoms status at the time the test was taken

3. You’ll need to collect the 11-digit barcode number of each test and the date and time that each test was taken. Retain a record of which barcode relates to which person.

4. You’ll need the consent of anyone you are registering a test kit for, to do so on their behalf.

  • Agreement should be documented in writing and obtained in line with your usual policies and procedures
  • Results should be returned to the persons you are registering test kits for using their email address and mobile phone number
  • However, you can get test results sent to you (as a manger) or a generic email address for your organisation (where access is restricted to you, management or key personnel), if you have written consent from the persons you are registering. It is then your responsibility to communicate the results of each test in a timely manner

Multiple registration upload process overview

1. Download the multiple upload spreadsheet

2. Pre-populate the personal details. Create separate master spreadsheets of up to 100 staff members and 100 non-staff members, and save these to be re-used every time you register tests. When individuals are not being tested on a particular day, their details should be removed from the spreadsheet and a new, dated copy saved for upload that day

3. Conduct testing. You are now able to input the details of tests kits directly into the spreadsheet. You no longer need to go back at the end to enter the test kit details in the portal:

  • use the drop-down list in column A to choose the test type
  • column B can be inputted by typing in the barcode number or by scanning in using the handheld scanner provided to you by NHS Test and Trace
  • if registering a rapid lateral flow test, you can report the test result in column G
  • If you select PCR as the test type in column A, the spreadsheet will automatically grey out the ‘test result’ in column G

4. You must enter a new row of data for each individual test being registered. You can register both PCR and rapid lateral flow tests for the same staff member on the same spreadsheet. You must input each test on its own row

5. Upload the saved electronic copy of the pre-populated record-keeping spreadsheet onto the online portal

6. A new screen is introduced allowing you to check the details of rapid lateral flow and PCR tests. Error messages will appear if a user updates the ‘Test kit barcode’ field with the wrong type of test barcode

7. Confirmation page and an email confirmation sent. The service now sends a confirmation email that summarises test kits registered. This email will be sent to the email address we have on file for your account. If you do not have an email address on file, no confirmation email will be sent. If you do not think we have the right email address for your organisation, please call 119

Step 7 refers to confirmation emails only and does not refer to test results, which will still be sent to the email and phone number provided during the registration process.

Pre-populating the multiple upload spreadsheet

To get the most benefit from using multiple upload to register more than one test kit at once and maximise the time saved on every testing day, you should pre-populate the spreadsheet with your staff and residents’ personal details.

You may want to print a copy of each complete spreadsheet for ease of record-keeping. However, all required information can now be inputted directly into the spreadsheet.

You can use the handheld scanners provided to you by NHS Test and Trace to further speed up the registration process by scanning test kits barcodes directly into the spreadsheet.

Tips

Once master spreadsheets have been created and saved, the personal information they contain can be re-used every time you test to speed up the registration process.

On each subsequent testing day, open the master spreadsheets and created new, dated copies of each. Remove the personal details of any person not being tested on that day, and save the dated copy so it can be uploaded to the organisation registration portal to register completed test kits. Save a copy for your records.

When conducting both types of testing on the same day for the same person, you’ll need to create 2 separate rows on the spreadsheet – one for the PCR test and the other for the rapid lateral flow test.

You’ll have to remember to keep the master spreadsheet up to date by adding or removing people as they join or leave your organisation.

Things to remember when populating the spreadsheet

All tabs can now be populated on the spreadsheet directly. You can also use the handheld scanners to scan tests barcodes directly into the spreadsheet.

When filling out the postcode field, you must enter the residential postcode of the person being tested, not the postcode of the organisation (unless the person permanently resides at the organisation’s address). This postcode is used to match GP records and is an NHS mandated requirement.

When populating the spreadsheet please ensure that it:

  • is complete and accurate for all individuals being registered
  • only contains details for up to 100 people at a time
  • is in the correct Excel file format (.xlsx)
  • does not contain any extra sheets, rows or columns, as this may impact the data upload

Uploading the spreadsheet to the portal

  1. Check the boxes to confirm that you have consent to register each person and for results to be sent to the contact details entered for them. Select ‘continue’

  2. Enter your 8-digit UON and select ‘continue’ (if you are a school or prison you’ll be taken down a different journey at this point)

  3. Check your organisation details and select ‘continue’

  4. For multiple registration, you’ll have to select the first option on this screen (‘use our spreadsheet to add a list of patients’). Select ‘continue’

  5. Select whether you are registering staff or non-staff and select ‘continue’

  6. Select ‘choose file’ to upload your completed record-keeping spreadsheet and once uploaded, select ‘upload’ to proceed

  7. Check rapid lateral flow tests details. An error message will appear if you update the ‘test kit barcode’ field with a PCR test kit barcode. To see details in full, select ‘view personal details’ under each name. Confirm the details are correct, then press ‘save and continue’

  8. Check PCR test details. An error message will appear if you update the ‘test kit barcode’ field with a rapid lateral flow test kit barcode. To see details in full, select ‘view personal details’ under each name. Confirm the details are correct, then press ‘save and continue’

  9. Check that the summary details are correct before selecting ‘save and continue’ to register your test kits

  10. Complete the security check and submit your registration by selecting ‘continue’. You’ll be unable to amend any details after ‘continue’ has been selected

  11. Your tests will now have been registered and a confirmation page will appear. You can select to register more test kits if needed

  12. A summary email will be sent to the email address NHS Test and Trace has on record for your organisation

Where you can access support

If you need help please contact the National Testing Programme contact centre on 119.

Look out for bulletin updates from organisation.coronavirus.testing@notifications.service.gov.uk.

Introduction

This guide is to help users understand what to do with any data adjustments we suggest in the LBCT return.

Each year the DfE reviews the content of the LBCT data to make sure it’s correctly recorded. There’ll be instances where we find inconsistencies in the data provided and we will ask you, through ‘data adjustments’, to review and, if you agree, correct these. This data adjustments guide is split into sections to show you how to make the changes in your form.

If you have any adjustments, you will access them via the ‘data adjustment’ section in your academy or trust index screen. In the example below, the adjustment is at trust level.

You may wish to download the data adjustments page as a reference point when reviewing and updating your LBCT return.

If you can’t see the ‘data adjustments’ section at the top of either your trust or academy index page, you won’t have any.

We have used our fictitious academy trust, Coketown, to demonstrate how to correct your occupancies where we have asked you to add, amend or delete them. We have also covered what to do if you disagree with our suggested data adjustments.

Need to add an occupancy

Scenario 1 - need to add an occupancy

In this example, the trust hasn’t added the sixth form area as an occupancy within the form. The data adjustment section in the academy index page will look like this.

Occupancy description Field adjustment relates to Suggested entry on return Suggested action Reason for adjustment
Sixth form area Occupancy type Leasehold Add missing occupancy route as follows: occupancy description sixth form area and land registry title deed ‘ABC12345’ Our review of the academy site has identified an additional occupancy that needs to be added to your LBCT return

Action needed

To do this, in the section entitled ‘Check your occupancies’ (as shown below), click on ‘Add a new occupancy’. Fill in the required fields based on the suggested action in the data adjustment table. You will also need to add any assets for the occupancy. You can read more about assets in our main guidance. In the case of Coketown, the only asset within the sixth form area (occupancy 3 on the map) is the land for the site. On the ‘add assets’ page, click on ‘add new asset’ and complete the follow-up questions for a land asset.

Need to change an occupancy description or type

We may ask you to change either an occupancy description, occupancy type or both. In this example, we will look at both.

The data adjustments will look like this.

Occupancy description Field adjustment relates to Current entry Suggested entry on return Suggested action Reason for adjustment
Main school site and woodland area Occupancy description Main school site and woodland area Main site Rename occupancy Our review indicates that the name of the occupancy is incorrect
Main school site and woodland area Occupancy type Leasehold Freehold Please change the occupancy type to freehold Our review indicates that the occupancy route is freehold

Action needed

To do both, navigate to the main school site and woodland area occupancy and amend the ‘Description of the site’ and ‘Occupancy type’ boxes – as shown below. If you make a change to some important fields, we will ask you to tell us the reason why you are making it. Select ‘suggested DfE adjustment’ from the dropdown of the question that asks for a reason for the change. Remember to review the rest of the section of the form as there may be additional questions to complete.

Need to amend data within an occupancy

As part of data adjustments to amend some important fields like ‘occupancy type’ as in example 2 above, we may also ask you to add or amend other information within your occupancies. For example:

  • landlord name
  • land Registry deed of register number
  • length of lease years
  • monthly rental payment amount
  • peppercorn

The data adjustments will look like this.

Occupancy description Field adjustment relates to Current entry Suggested entry on return Suggested action Reason for adjustment
Main site Land registry deed number NN123456 NN987654 Update title deed number Our review indicates that the title deed number is incorrect
Sixth form area Landlord Mr Smith Coketown District Council Update landlord name Our review indicates that the landlord name is incorrect
Sixth form area Monthly payment amount 0 250 Update the monthly payment amount Our review of the leasehold agreement shows a monthly payment amount

Action needed

To do this, simply go to the relevant field and make the requested changes as you work through the appropriate occupancy. Remember that for some important fields we ask for a reason for making the change. All data adjustment requests from the DfE should provide the reason ‘Suggested DfE adjustment’.

Need to delete an occupancy

We may ask you to delete an occupancy if:

  • it’s been entered in error (usually because it has or should be recorded as an asset)
  • it’s a duplicate

When an occupancy is deleted, all associated assets and documents will also be removed. You should therefore ensure that you save any documents and information that might be needed for further completion of the form before you delete the occupancy.

Worked example

In this scenario, the trust has incorrectly reported the arts and drama block as an occupancy as well as an asset within another occupancy (the latter being correct).

The data adjustment page will look like this.

Occupancy description Field adjustment relates to Current entry Suggested entry on return Suggested action Reason for adjustment
Arts and drama block Occupancy type Freehold - Delete this occupancy Our review indicates that this has been added as an occupancy when it should have been added as an asset instead. This is already included within another occupancy already added to the form

Action needed

Find the arts and drama block occupancy in the list of occupancies and select ‘remove occupancy’.

From the drop-down list that appears, choose ‘removal of invalid occupancy’ followed by ‘delete occupancy’. Once done, this action cannot be reversed or undone.

The page will update and the user should click ‘save and back to academy index’.

What if I disagree with the suggested data adjustments?

All users can see and make data adjustments but only the approver can sign off the declaration to accept or reject them.

The approver will be asked to confirm if they agree or disagree with the data adjustments via a declaration statement. If they disagree, they will need to explain why.

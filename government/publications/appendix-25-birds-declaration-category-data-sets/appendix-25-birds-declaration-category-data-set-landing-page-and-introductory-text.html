This Appendix provides a detailed breakdown of the specific data elements needed for the Bulk Import Reduced Data Set (BIRDS) declaration category.

The Appendix is broken into 2 parts:

  • Appendix 25: Declaration Category Data Set Landing Page and Introductory Text
  • Appendix 25A: Declaration Category Data Set BIRDS

1. Description of the Declaration Categories covered by Appendix 25 BIRDS

1.1 Appendix 25A: BIRDS

BIRDS — Bulk Import Reduced Data Set

2. General Notes

Appendix 25: Declaration Category Data Set BIRDS contains a table for the BIRDS declaration category. The table also provides a cross reference from the new data element number to the previous SADH box reference number for correlation purposes.

The data set table also states whether the data element is mandatory in all cases, optional for the declarant or dependant on the circumstances being declared.

The Data Set table identifies the requirements for completion of the appropriate data elements on a BIRDS declaration.

A consolidated set of Notes are provided to assist in reading the data set table.

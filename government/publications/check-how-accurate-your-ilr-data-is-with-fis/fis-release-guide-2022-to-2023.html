1. Introduction

Funding Information System (FIS) is an offline tool you can use to check your Individualised learner data (ILR) against the validation rules before making an ILR submission.

2. FIS functions:

  • Import and validate an ILR submission
  • Perform funding calculations
  • Export ILR data as: CSV files, SQL files or to an Access database

3. User advice

  • All versions of FIS have only been tested using the standard setup, any deviations from this setup are un-supported.
  • FIS is only compatible with SQL 2012 and subsequent versions.

4. Document purpose

This document provides a guide to supplement each release of the FIS. It describes the changes and other notable information about the desktop application across the 2022 to 2023 funding year for which FIS is active.

This document does not provide user guidance for the whole application. Please see the relevant documentation on the FIS user guide for 2022 to 2023 which can be found on GOV.UK.

5. 2022 to 2023 version history

FIS can be downloaded from Submit Learner Data.

The release with the highest version number will always be the latest version available. Unless there is a relevant Known Issue, users are advised to select this version when processing ILR data to achieve the most up to date results from FIS.

*Every release of FIS will contain up to date reference data which has also been loaded to Submit Learner Data this includes Funding Calculations, FALA (previously called LARS) and Organisation data updates

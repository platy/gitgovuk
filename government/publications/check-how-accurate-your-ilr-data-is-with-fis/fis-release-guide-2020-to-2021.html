1. Introduction

Funding Information System (FIS) is an offline tool you can use to check your Individualised learner data (ILR) against the validation rules before making an ILR submission.

2. Document purpose

This document provides a guide to supplement each release of the FIS. It describes the changes and other notable information about the desktop application across the 2020 to 2021 funding year for which FIS is active.

This document does not provide user guidance for the whole application. Please see the relevant documentation on the FIS user guide for 2020 to 2021 which can be found on GOV.UK.

3. User advice

All versions of FIS have only been tested using the standard setup, any deviations from this setup are un-supported.

FIS is only compatible with SQL 2012 and subsequent versions.

Please see the feedback section at the end of this document for instructions on how to report any associated issues.

4. 2020 to 2021 version

FIS can be downloaded from Submit Learner Data.

The release with the highest version number will always be the latest version available. Unless there is a relevant Known Issue, users are advised to select this version when processing ILR data to achieve the most up to date results from FIS.

  • Every release of FIS will contain up to date reference data which has also been loaded to Submit Learner Data this includes Funding Calculations, LARS and Organisation data updates

7. Known issues

Whilst we try to ensure FIS is accurate, if any issues are identified they will be captured in the ‘FIS known issues 2020 to 2021’ document referring to the ‘Active’ tab. This document is available on GOV.UK.

8. Feedback

If you have a question regarding FIS and its’ functionality then please contact the Service Desk on:

  • Email: SDE.Servicedesk@education.gov.uk
  • Telephone: 0370 2670001

Before reporting issues, please ensure that you are using the latest available version of FIS and have checked the known issues document as the issue may already be identified.

What we’re working on

Find out about the work we are doing to improve the service we provide via the HM Land Registry portal and Business Gateway platform.

We list recent changes customers can use, projects we are actively working on and ideas we are exploring for the future.

April 2022 update

Recently added

HM Land Registry portal

  • Amendment to portal home page to add in hours of service

Digital Registration Service ​

Release sets of enhancements to the service in the portal, which include:​

  • updates to the discharges journey which streamline the journey​
  • updates to the ‘complete your application page’​
  • updated in service banner which provide a view of all changes made to the service​
  • removal of attachment checklist based on customer feedback

Search for land and property ​information

  • Users now the option of a title plan view​
  • Switched off the Find a property service so there is now a single service for citizens to view free and paid for property information

Business Gateway​

  • Registered Proprietor Name API released into the live environment which allows case management systems to develop against it​

  • Launched new styling of the developer pack which supports easier integration with Business Gateway

General ​

  • ​Further lenders added to the ‘Sign your mortgage deed’ service

Working on now

Digital Registration Service​

  • Creating a single point of entry in portal Digital Registration Service for all register change interactions

Business Gateway ​

  • Opening up the API gateway to enable new API management tools providing service specific controls​
  • Building ‘OS1 as data’ API for users – enabling them to receive the OS1 result as data as well as PDF

HM Land Registry portal ​

  • Extending ‘View My Applications’ to enable users to search for and access colleagues’ applications​ ​* Development of a new home page which will add personalisation and easier navigation

Search for land and property ​information

  • Increasing the search options for users by developing search by map

Exploring

General​

February 2022

Progress made

  1. Portal users now have opt in/opt out functionality in readiness for the view colleagues’ applications feature​.
  2. Saved Digital Registration Service applications can now be accessed from View My Applications in the portal, providing an improved user experience​.
  3. Implemented the full flow-through journey for transfers, assents and discharge transactions, which means it is quicker and easier to complete an application via the Digital Registration Service in the portal​.
  4. Improved in-journey navigation, including easy access to the ‘complete your application page’ for all transactions in the Digital Registration service.
  5. Added more informative and directive error messaging for all transactions in the Digital Registration Service.
  6. Users now have the option of a title plan view in Search for land and property ​information​.
  7. All of our business channels have been updated to handle changes to fees from 31 January​.

December 2021

Progress made

  1. Our Digital Registration Service now accepts applications that include power of attorney.
  2. Citizens can use our Find a property service to search for a property using an Inspire ID.
  3. Citizens can now create an account to access our Search for land and property information service. This allows us to introduce more features in future releases (includes paid services).
  4. Providing citizens with a register view in Search for land and property information. This allows citizens to find out what a current owner paid for their property and download a title register.
  5. Pushed the ‘Document Availability’ Business Gateway application programming interface (API) into the test environment.
  6. Portal users can now leave their accounts inactive for 18 months before we close them.
  7. Provided the ‘my documents’ section in Search for land and property information which hold paid for information for 36 hours.
  8. Pushed the ‘Name Discrepancy’ Business Gateway API service into the test environment.

October 2021

Progress made

  1. A pilot for receiving qualified electronic signatures and the first one being received.

  2. Multiple title register update applications can now be lodged through the Digital Registration Service in the portal.

  3. Increased the potential users for the Sign my mortgage deed service by onboarding further lenders.

  4. Increasing speed of service through the Sign my mortgage deed service through changing some original criteria.

  5. Pushed the ‘OC document availability’ Business Gateway API into the test environment.

  6. Launched an interactive public backlog for Business Gateway users to vote on what enhancements they want to see in the channel.

  7. Further development of the ‘view my applications’ feature in the portal.

  8. Provided quality and better structured register data to enable the registered proprietor name to be validated pre submission in the Digital Registration Service in the portal.

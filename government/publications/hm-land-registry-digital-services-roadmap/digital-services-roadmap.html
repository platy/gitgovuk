What we’re working on

Find out about the work we are doing to improve the service we provide via the HM Land Registry portal and Business Gateway platform.

We list recent changes customers can use, projects we are actively working on and ideas we are exploring for the future.

February 2022 update

Recently added

HM Land Registry portal

  • Users have opt in/opt out functionality in readiness for the view colleagues’ applications feature​.
  • Saved Digital Registration Service applications can now be accessed from View My Applications in the portal, providing an improved user experience​.

Digital Registration Service ​

  • Implemented the full flow-through journey for transfers, assents and discharge transactions, which means it is quicker and easier to complete an application in the portal​.
  • Improved in-journey navigation, including easy access to the ‘complete your application page’ for all transactions.
  • Added more informative and directive error messaging for all transactions.

Search for land and property ​information

  • Users now have the option of a title plan view​.

General ​

  • All channels updated to handle changes to fees from 31 January​.

Working on now

Pilot

  • Extending the scope for the pilot of qualified electronic signatures to more users​.

Digital Registration Service​

  • Completing the register update application set and enhancing existing service journey and features​.

Business Gateway ​

  • Development of developer pack to support easier integration with Business Gateway.​
  • ​Opening up the API (application programming interface) gateway to enable new API management tools providing service-specific controls​.
  • Developing the concept of OS1 as data for users​.

HM Land Registry portal ​

  • Extending View My Applications to enable users to search for and access colleagues’ applications.​

Search for land and property ​information

  • Increasing the search options for users by developing search by map.

General​

  • Increasing the file size for variable direct debit file sizes for information service requests.​

Exploring

HM Land Registry portal

  • Further portal channel features which will increase self service, including a new home page design and consolidation of services​.

Business Gateway​

  • Analysis on the services and features needed to develop end-to-end solutions for digital registration for users.
  • Building of a developer hub to support easier integration with​ our APIs and/or services.

Search for land and property​ information

  • Building out the initial service so it is the only place citizens will need to go to view property information.
  • Extending the service to enable ordering official copies and property monitoring.​

General​

  • Understanding all data requirements to enable further data to be made available to users​.
  • Understanding how speed of service can be increased across register updates through the use of technology.
  • Planning the switch off for Find a property so there will be a single service for citizens to view free and paid for property information​.

December 2021

Progress made

  1. Our Digital Registration Service now accepts applications that include power of attorney.
  2. Citizens can use our Find a property service to search for a property using an Inspire ID.
  3. Citizens can now create an account to access our Search for land and property information service. This allows us to introduce more features in future releases (includes paid services).
  4. Providing citizens with a register view in Search for land and property information. This allows citizens to find out what a current owner paid for their property and download a title register.
  5. Pushed the ‘Document Availability’ Business Gateway application programming interface (API) into the test environment.
  6. Portal users can now leave their accounts inactive for 18 months before we close them.
  7. Provided the ‘my documents’ section in Search for land and property information which hold paid for information for 36 hours.
  8. Pushed the ‘Name Discrepancy’ Business Gateway API service into the test environment.

October 2021

Progress made

  1. A pilot for receiving qualified electronic signatures and the first one being received.

  2. Multiple title register update applications can now be lodged through the Digital Registration Service in the portal.

  3. Increased the potential users for the Sign my mortgage deed service by onboarding further lenders.

  4. Increasing speed of service through the Sign my mortgage deed service through changing some original criteria.

  5. Pushed the ‘OC document availability’ Business Gateway API into the test environment.

  6. Launched an interactive public backlog for Business Gateway users to vote on what enhancements they want to see in the channel.

  7. Further development of the ‘view my applications’ feature in the portal.

  8. Provided quality and better structured register data to enable the registered proprietor name to be validated pre submission in the Digital Registration Service in the portal.

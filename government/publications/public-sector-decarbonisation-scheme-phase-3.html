Details

Phase 3 of the Public Sector Decarbonisation Scheme is providing £1.425 billion of grant funding over the financial years 2022/2023 to 2024/2025. It supports the aim of reducing emissions from public sector buildings by 75% by 2037, compared to a 2017 baseline, as set out in the Net Zero and Heat and Buildings strategies.

There will be multiple application windows for Phase 3 of the scheme. The first Phase 3 application window, Phase 3a, is closed to applications. The guidance for the next application window, Phase 3b, will be published in July, with the application window planned to open in September 2022.

For more information about Phase 3, please see the Salix Finance website.

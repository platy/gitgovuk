Details

These documents give technical information about submitting data for the 2021 to 2022 children’s social work workforce census. They describe the data items for collection in the census.

These documents are for software suppliers and users of local authority management information systems.

You should also read the common basic data set (CBDS). This defines common data items used in management software and what we use in our data collections.

More guidance on submitting data for the children’s social work workforce census is available.

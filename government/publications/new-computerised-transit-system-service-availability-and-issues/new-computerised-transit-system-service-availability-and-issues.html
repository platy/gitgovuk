Service issues

Decommissioning of the NCTS Legacy XML channel

Work has started on the closure of the Legacy XML channel for NCTS 2. This will run until 11:59pm on 13 March 2022, after which you will no longer be able to make a new declaration using the channel.

You will be able to do status checks on in-flight movements on the NCTS Legacy XML channel for a further 30 days, however there will be a brief maintenance period on the NCTS Legacy XML channel from 11:59pm 13 March 2022 to 7am 14 March 2022.

If you use the NCTS email channel, the final date to make declarations will be 31 May 2022, when we will also close this channel to new declarations.

You must move to the new XML Application Programming Interface (API) channel to help manage post-Brexit volumes. You should speak to your own software provider to make sure they’re ready for this transition.

If you cannot move to the new XML API, you should submit by using either the free to use web portal or non-transit customs processes after these channels are closed.

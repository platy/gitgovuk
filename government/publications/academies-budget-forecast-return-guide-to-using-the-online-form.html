Details

The academies budget forecast return (BFR) is now live. The deadline for the BFR to be submitted is 26 July 2022. To access the form use the link below:

BFR Login

Service notices

In the event that the form doesn’t load after clicking the BFR tile, we recommend either clearing your cookies and/or using a private/incognito window.

We have published the guide for the academies budget forecast financial return - BFR. This guide is to be used to help all users complete the financial return. For more details please see the main budget forecast overview page - this includes the excel workbook to help you prepare for your return.

Questions

If you have any questions about academies budget forecast returns, please contact us using our online enquiry form below. Please put ‘academies budget forecast returns’ at the top of your enquiry so we can quickly direct your question to the team.

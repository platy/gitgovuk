Details

From 1 September 2022, you can use a CSV file to upload commodity details to the Import of products, animals, food and feed system (IPAFFS).

You must enter exact CSV values. Use the CSV values for IPAFFS document to find out how to enter information into the CSV file.

Get help

If you need technical help uploading commodity details using a CSV file:

  • call the Animal and Plant Health Agency (APHA) helpline on 0330 041 6999
  • email APHAServiceDesk@apha.gov.uk

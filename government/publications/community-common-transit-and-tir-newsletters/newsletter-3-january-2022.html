Changes to the email and legacy Extensible Markup Language (XML) channels to access the New Computerised Transit System (NCTS) at the end of this month

Access to the legacy XML channel for NCTS will close on 28 February 2022

We have changed our previously published intention to close the email channel on the same date. This will now take place on 31 May 2022.

In our earlier communications, found here; Community, Common Transit and TIR newsletters - GOV.UK (www.gov.uk), we explained the need to move to the new XML API (Application Programming Interface) channel to help manage expected post-Brexit volumes.

We are aware of challenges for software developers and traders in meeting this deadline for email and legacy XML decommissioning.

Following feedback and a review of readiness with both software developers and traders, we have taken the decision to extend the date on which we close the email channel to help enable transition to the XML API for users of email.

In extending the date to 31 May 2022, we have taken into account the challenges faced and the busy time period we have had at the end of 2021.

This extension applies only to the email channel, the Legacy XML Channel will still close on 28 February 2022

This will be the case for all Great Britain (England, Scotland and Wales) and Northern Ireland movements.

To reiterate:

  • although we have delayed the date for closing the email channel, you still need to migrate to the new XML API as soon as possible
  • you may be able to use email in the short term if you have the capability to do so, but you still need to update to the new XML API
  • if you cannot move to the new XML API for any reason, the free-to-use HMRC online portal is available and will give continuing access to the NCTS — however, you should note that this may not be ideal for large volume users

If your software currently uses the legacy XML or email channel to send messages, you must take action

Steps to take now if you are still using the legacy XML channel to access the NCTS

If you’re using the legacy XML channel to access the NCTS:

  • you need to check with your software provider now, what these changes mean for you and what action they need to take to be ready for you to use the new XML API Channel before 28 February 2022
  • you should also check with your software provider to ensure you have all the relevant information you may need to log in — for example, your Government Gateway credentials
  • you must ensure you have all necessary software upgrades and will be ready to use the new software package as soon as possible, as you will not be able to make any declarations using the legacy XML channel after 28 February 2022
  • if you are not ready by 28 February 2022, the alternative to the new XML API channel is the free-to-use HMRC online portal
  • if you have access to the email channel, this can continue to be used until 31 May 2022, but it will then also be closing

Steps to take now if you are still using the email channel to access the NCTS

If you’re still using the email channel to access the NCTS:

  • you need to check with your software provider now, what these changes mean for you and what action they need to take to be ready for you to use the new XML API Channel before 31 May 2022
  • you should also check with your software provider to ensure you have all the relevant information you may need to log in — for example, your Government Gateway credentials
  • you must ensure you have all necessary software upgrades and will be ready to use the new software package as soon as possible, as you will not be able to make any declarations using the email channel after 31 May 2022
  • if you are not ready by 31 May 2022, the only alternative to the new XML API will be the free-to-use HMRC online portal

If you wish to enrol for the New Computerised Transit System web portal using Government Gateway, please ensure you do so in advance to gain full access. This enrolment and activation process can take up to 2 weeks or more.

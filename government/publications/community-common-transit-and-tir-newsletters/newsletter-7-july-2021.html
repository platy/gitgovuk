Changes to the XML channel to access the New Computerised Transit Service (NCTS)

If you access the NCTS using the XML channel to submit transit movements, you will be using commercial software.

We have been working with software developers over the past 12 months, to develop a new XML channel which will:

  • mean a more stable service
  • reduce the potential for unexpected outages like those recently experienced
  • be able to deal with the expected increase in volumes as a result of EU Exit

That new channel is now available, and we are working with software developers to make the necessary changes to their software packages so they work with the new channel. Many of these changes are not visible and will not affect the way you submit your transit movements.

As part of this work we intend to close the legacy XML channel. The current plan is to do that on 30 November 2021, and we told software developers about this on 1 June 2021.

After this date you will need to use either:

  • software compatible with the new XML API channel
  • the free-to-use HMRC online portal

This will be the case for all Great Britain (England, Wales and Scotland) and Northern Ireland movements.

Using the XML channel to access the NCTS

Before 30 November 2021, you will need to check with your software provider what these changes may mean for you.

Using the email channel to access the NCTS

You can continue to use this method to access the NCTS but we will be decommissioning this channel in due course.

As part of developing the timeline for that decommissioning work we are carrying out some consultation to understand current usage of the email channel and to encourage usage of the new XML channel.

Once we have more information regarding timeframes for the email decommissioning we will share this with you.

Changes to the XML channel to access the New Computerised Transit Service (NCTS)

We have been working with software developers to introduce a new XML channel that will:

  • help make the system more robust by reducing unexpected outages
  • deal with the expected increase in volumes as a result of Brexit and the anticipated increased instability in the legacy channel

The new XML channel has been live since 31 March 2021 in Great Britain, and 13 May 2021 in Northern Ireland.

This will not affect the way you submit your transit movements. Your software provider should be able to tell you more about their plans.

We have also contacted software developers to remind them of the need to ensure you are able to move to the new XML API channel. As part of this work we plan to close the legacy XML channel on 30 November 2021.

After the 30 November 2021, you’ll need to use either:

  • software compatible with the new XML API channel
  • the free-to-use HMRC online portal

This will be needed for all Great Britain (England, Wales, Scotland and Northern Ireland) movements. If your software currently uses the legacy XML channel to send messages, you must take action now.

Steps to take if you are using the XML channel to access the NCTS

You need to check with your software provider, what these changes may mean for you and if you need to take any action. You should also check with your software provider to ensure you have all the relevant information you may need to log into your Government Gateway account for example.

If you need to update your software package to be able to use the new XML API channel, you must do this before 30 November 2021.

It is vital that you have made any necessary software upgrades and will be ready to use the new software package, as you will not be able to make any declarations using the current legacy XML channel after 30 November 2021.

Registration

If you’re not currently registered to use NCTS you’ll need to allow enough time for your registration to be processed.

Using the email channel to access the NCTS

You can continue to use the email channel to access the NCTS after 30 November 2021. However, we will be decommissioning this channel in due course and will confirm that timeline shortly.

You should consider how you will seek to make declarations once the email channel is decommissioned and speak to your software provider if you have one already, as there will be no replacement for the email channel.

As part of developing the timeline for the decommissioning work, we are encouraging the use of the new XML channel for which you will need to speak to your provider or contact one who can provide access. The alternative is the free-to-use HMRC online portal.

Pubs Code and Pubs Code Adjudicator: invitation for views on the second statutory review, 2019 to 2022

From:
Department for Business, Energy & Industrial Strategy
Published
26 May 2022

Summary

We invite views and evidence on the operation of the Pubs Code and the performance of the Pubs Code Adjudicator which, as part of the second statutory review, will inform its conclusions.

This consultation closes at
11:45pm on 17 August 2022

Consultation description

The Small Business, Enterprise and Employment (SBEE) Act 2015 provided for the establishment of the Pubs Code which regulates the tenancies of large estates of tied pubs in England and Wales. The Act and the Regulations require the Secretary of State to regularly review the operation of the Pubs Code and certain other provisions, and the Adjudicatorâ€™s performance. 

The second review covers the period from 1 April 2019 to 31 March 2022.

We are seeking views from all those with an interest in the operation of the Pubs Code and the performance of the Pubs Code Adjudicator. This includes, but is not restricted to:

  • the pub-owning businesses covered by the Code
  • tied (and previously tied) pub tenants and those that represent their interests
  • trade bodies
  • the Pubs Code Adjudicator
